# Wedding Editorial Invitation - Final

GitHub Pages에 바로 업로드 가능한 모바일 청첩장 최종 버전입니다.

## 수정 위치
- 기본 정보: `config.js`
- 이미지: `images/` 폴더
- 음악: `music/bgm.mp3`

## 주요 기능
- 풀스크린 히어로 이미지
- 자동 생성 달력 + 결혼 날짜 표시
- 갤러리 전체화면 보기 / 좌우 넘김 / 모바일 스와이프
- 네이버 지도 미리보기 + 네이버/카카오/T맵 버튼
- 교통 안내 + 주차 안내
- 접힘형 애프터파티 안내
- 신랑측/신부측 계좌 아코디언 + 복사 버튼
- 구글드라이브 사진 업로드 버튼
- 카카오톡 공유 / 링크 복사
- 상단 고정 미니 메뉴

## 구글드라이브 사진 업로드 링크
`config.js`에서 아래 값을 수정하세요.

```js
photoUpload: {
  url: "여기에_구글드라이브_업로드_링크를_넣어주세요"
}
```

구글드라이브 폴더 공유 권한은 `링크가 있는 모든 사용자 - 편집자`로 설정해야 하객이 업로드할 수 있습니다.
